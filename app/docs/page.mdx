# Getting started

This guide walks you through installing osTicket using its built-in web installer. It’s designed for users with a basic understanding of web servers, PHP, and MySQL. By the end, you’ll have a fully functional osTicket instance up and running, ready for configuration and use.

## About osTicket

osTicket is a free, open-source helpdesk system that streamlines how organizations manage support requests. It centralizes tickets from multiple channels—like email, web forms, and phone calls—into a single, easy-to-use dashboard. Designed for teams of all sizes, osTicket helps automate assignments, track performance, and ensure timely responses through built-in features like SLAs, canned replies, and customizable workflows. Whether you’re setting up an IT helpdesk or customer support center, osTicket offers a powerful and flexible foundation to get started quickly.

## Prerequisites

Before starting, make sure your server meets the following requirements:

### Server Requirements

- **Web Server:** Apache, LiteSpeed, or IIS (with URL Rewrite enabled)
- **PHP:** Version 8\.1 – 8\.2 (8\.2 recommended)
- **Database:** MySQL 5\.0\+ or MariaDB 10\.x

If you’re unsure, contact your hosting provider or system administrator.

### Database Access

You’ll need a MySQL database with:
- a **valid user**, **password**, and **hostname**
- **FULL privileges** granted to that user

If unsure, ask your host or database admin before proceeding.
