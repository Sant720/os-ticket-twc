## Text Formatting

### Basic Formatting
```mdx
**Bold text** for emphasis

*Italic text* for slight emphasis

***Bold and italic*** for strong emphasis

~~Strikethrough~~ for deprecated content

`inline code` for code snippets
```

**Bold text** for emphasis

*Italic text* for slight emphasis

***Bold and italic*** for strong emphasis

~~Strikethrough~~ for deprecated content

`inline code` for code snippets

### Headers
```mdx
# H1 Heading
## H2 Heading
### H3 Heading
#### H4 Heading
##### H5 Heading
###### H6 Heading
```

```mdx
- First item

  - Nested item
  - Another nested item
- Third item
```

   2. Sub-step B

## HTML Elements

You can use HTML elements directly in MDX:

```mdx
<details>
  <summary>Click to expand</summary>
  
  This content is hidden until you click the summary.
  
  You can include any markdown here:
  - Lists
  - **Bold text**
  - Code blocks
</details>
```
